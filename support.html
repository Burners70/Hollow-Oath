Common questions

My progress didn't come back after reinstalling / on my other device.
Progress saves locally and, if you're signed into iCloud, syncs there automatically. Check Settings → [your name] → iCloud → make sure iCloud Drive (or "Show All Apps") is on for Hollow Oath, on both devices, then reopen the game — sync can take a minute. If it's still missing, email us; we'll ask what you last saw on the title screen so we can tell where the save stopped.
Game Center isn't showing my score / achievements.
Confirm you're signed into Game Center (Settings → Game Center) and that you allowed it when the game first asked. Scores and achievements are handled entirely by Apple's Game Center — we don't run our own leaderboard, so this is usually a sign-in state rather than a game bug.
I ran a Field Medic (easy mode) run — why didn't it post to the leaderboard?
That's intentional, not a bug — Field Medic runs are a story-focused mode and are deliberately excluded from the main competitive board so it stays a fair comparison. Play a standard run to post a leaderboard score.
Controls feel unresponsive.
Hollow Oath uses on-screen touch controls. If they feel off, tell us your device model — screen size and safe-area insets change where the buttons sit, and that's the single most useful thing for us to know when something feels wrong.
The game crashed, froze, or wouldn't launch.
Email us with your device model and iOS version. If you can reproduce it (e.g. "always happens when I open the codex"), that one line is the single most useful thing you can send.
I have a refund or purchase question.
Purchases and refunds are handled entirely by Apple, not us — request a refund at reportaproblem.apple.com or via Settings → [your name] → Media & Purchases → Purchase History on your device.

Something else / found a bug you can describe precisely

Email us at hollowoath.support@gmail.com — a clear description of what you did and what happened is all we need to start looking into it.
